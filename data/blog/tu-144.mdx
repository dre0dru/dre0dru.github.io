An interactive museum inside legendary supersonic aircraft.
This project is a flying simulator with physical control elements (steering wheel, pedals, tumblers, arrows, etc.) and dynamic suspension seat.
Aircraft control panel is emulated with 8 mini displays.

<video controls="controls" loop="loop" src="../static/videos/portfolio/tu144.mp4" />

Responsibilities: gameplay and scenario logic, flying physics, sound, sampling input from physical controls, sending data back (for highlighting control elements with diodes), controlling seat suspension.
TCP client/server for aircraft control panel data synchronization.

Technologies: Unity, C#, FMOD
